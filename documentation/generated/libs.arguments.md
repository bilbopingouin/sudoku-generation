# libs.arguments

This library handles the command-line user interface.

The following options are available

```default
usage: sudoku.py [-h] [-i INPUT] [-d]

Sudoku generator

options:
    -h, --help            show this help message and exit
    -i INPUT, --input INPUT
                          Input Sudoku String
    -d, --debug           Activate some output debug
```
